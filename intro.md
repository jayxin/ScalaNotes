# Scala

- Scala 基于 JVM, 和 Java 完全兼容, 同样具有跨平台、可移植性好、方便的垃圾回收等特性;
- Scala 比 Java 更加面向对象;
- Scala 是一门函数式编程语言;
- Scala 更适合大数据的处理, 对集合类型数据处理有非常好的支持
- Spark 的底层用 Scala 编写

## 主要内容

- 基本语法
    1. 变量和数据类型
    2. 运算符
    3. 流程控制
- 核心特性
    1. 函数式编程
    2. 面向对象
    3. 集合
- 其他
    1. 模式匹配
    2. 异常处理
    3. 隐式转换
    4. 泛型
